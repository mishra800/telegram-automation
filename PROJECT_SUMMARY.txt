✓ Generates AI content using local LLM (Ollama)

================================================================================
AUTOMATION FLOW
================================================================================

1. Scheduler triggers at configured time
2. System selects topic based on performance weights
3. Ollama generates content (or uses fallback)
4. Stable Diffusion creates image (or gradient)
5. Bot posts to Telegram channel/group
6. Post metadata saved to database
7. After 24 hours, stats collected
8. Weekly analysis adjusts topic weights
9. Future posts favor high-performing topics
