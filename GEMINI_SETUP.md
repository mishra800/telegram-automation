## ЁЯФС Get Your FREE Gemini API Key

### Step 1: Go to Google AI Studio
Visit: https://makersuite.google.com/app/apikey

### Step 2: Sign In
- Use your Google account
- Accept terms of service

### Step 3: Create API Key
1. Click **"Get API Key"** or **"Create API Key"**
2. Select **"Create API key in new project"**
3. Copy the API key (looks like: `AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX`)

### Step 4: Update .env File

Open your `.env` file and update:

```env
# Enable Gemini
USE_GEMINI=true
GEMINI_API_KEY=AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
```

Replace `AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX` with your actual API key.

### Step 5: Install Gemini Package

```bash
pip install google-generativeai
```

Or reinstall all requirements:
```bash
pip install -r requirements.txt
```

### Step 6: Test It

```bash
python main.py test-content motivational
```

You should see content generated by Gemini Pro!

## ЁЯОп How It Works Now

### Priority Order:
1. **Gemini Pro** (if enabled and API key provided)
2. **Ollama** (if Gemini fails)
3. **Fallback content** (if both fail)

### Configuration Options:

**Use Gemini only:**
```env
USE_GEMINI=true
GEMINI_API_KEY=your_key_here
```

**Use Ollama only:**
```env
USE_GEMINI=false
```

**Use both (Gemini first, Ollama backup):**
```env
USE_GEMINI=true
GEMINI_API_KEY=your_key_here
OLLAMA_MODEL=llama3
```
